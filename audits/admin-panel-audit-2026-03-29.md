## Технические заметки (не критично)

- **GitHub PAT в localStorage** — при включённом "Запомнить на устройстве" токен хранится в localStorage. Есть опция sessionStorage (чекбокс выключен). Для общих устройств рекомендуется не запоминать.
- **Firebase config публичен** — ожидаемо для Firebase Auth. Важно настроить правила безопасности в Firebase Console (допустить только конкретные email-адреса).
- **Rate limiting** — кнопка Publish блокируется через `btnLoading()` на время запроса, но теоретически можно исчерпать GitHub API лимит (5000 req/hr) при массовых операциях.
