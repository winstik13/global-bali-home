# CLAUDE.md

Этот файл содержит инструкции для Claude Code (claude.ai/code) при работе с кодом в этом репозитории.

## Обзор проекта

Global Bali Home — многостраничный маркетинговый сайт премиального девелопера недвижимости на Бали. Чистый HTML/CSS/JS, без фреймворков и сборщиков.

## Архитектура

- **9 HTML-страниц** в корне (EN) + зеркала в `ru/` (RU), `id/` (Bahasa Indonesia), `zh/` (简体中文) — **36 страниц всего**. В шапке — выпадающий переключатель языков (EN ▾), показывающий все 4 языка.
  - Страницы: `index.html` (главная), `about.html`, `projects.html`, `services.html`, `gallery.html`, `contacts.html`, `project-serenity-villas.html`, `project-serenity-estates.html`, `project-serenity-village.html`
  - Коды языков: `en` (корень), `ru/`, `id/`, `zh/`
- **`css/style.css`** — вся дизайн-система: CSS-переменные, тёмная тема, все компоненты, responsive breakpoints (1024px, 768px)
- **`css/reset.css`** — стандартный CSS reset
- **`js/main.js`** — вся интерактивность: скролл шапки, бургер-меню, scroll reveal (IntersectionObserver), параллакс, счётчики чисел, динамический рендер галереи, лайтбокс, FAQ аккордеон, валидация форм, квиз-попап, ROI калькулятор, exit intent попап, sticky CTA бар, лид-магнит форма
- **`gallery-data.js`** — автогенерируемый манифест изображений галереи, подключается как `<script>` в `gallery.html` и `index.html`
- **`generate-gallery.ps1`** — PowerShell-скрипт, сканирует папки изображений и генерирует `gallery-data.js`
- **`Base info.md`** — справочный файл с актуальными данными проектов (кол-во вилл, спальни, цены, даты сдачи, статусы юнитов). Сначала обновлять этот файл, затем синхронизировать изменения в HTML.

### JS-инжектируемые компоненты (DOM создаётся в main.js, нет дублирования в HTML)

- **Квиз-попап** — 3-шаговая квалификация лида (цель → бюджет → сроки → рекомендация + контактная форма). Вызывается атрибутом `[data-quiz]` или кнопкой `.cta-section .btn--primary`. Показывает рекомендацию проекта ДО формы контактов — снижает abandonment.
- **Exit intent попап** — срабатывает на `mouseleave` (только десктоп) после 30 сек, подавляется если квиз открыт или лид уже захвачен. Использует `sessionStorage` (`exitShown`, `leadCaptured`).
- **Sticky CTA бар** — фиксированная нижняя панель только на мобильных, появляется после скролла мимо hero, открывает квиз.

```

```

### Рабочий процесс галереи

Изображения галереи загружаются динамически из `gallery-data.js`. После добавления/удаления изображений в папках проектов — перегенерировать манифест:
```
powershell -ExecutionPolicy Bypass -File generate-gallery.ps1
```

## Ключевые конвенции

- **Тёмная премиум-тема**: фон `#1a1a14`, текст `#E1D9C9`, акцент `#6B8F4E`
- **Шрифты**: Playfair Display (заголовки через `--font-heading`), Montserrat (текст через `--font-body`), загружаются через `<link>` из Google Fonts CDN с `preconnect`
- **CSS-классы для scroll-анимаций**: `.reveal`, `.reveal-left`, `.reveal-right`, `.reveal-stagger` — срабатывают через IntersectionObserver, добавляющий `.visible`
- **Счётчик чисел**: добавить атрибут `data-counter` к элементам с числовым текстом
- **Фон секций**: по умолчанию `--color-bg`, использовать `.bg-alt` для альтернативных тёмных секций. Последовательные секции должны чередовать цвета, чтобы не сливаться.
- **Fullbleed hero** (`.fullbleed-hero`): используется на about, contacts, services и 3 страницах проектов. Фоновое изображение плавно переходит в контент через градиентный оверлей. Структура: `__bg` (фоновое изображение) + `__overlay` (градиент) + `__top` (хлебные крошки, подзаголовок, h1) + опционально `__bottom` (первая секция контента, вмержённая в hero). Заменяет `.page-hero` на этих страницах.
- **Inline hero-stats** (`.hero-stats`): компактная горизонтальная строка статистики под h1 на страницах проектов (элементы через вертикальную черту). Заменяет старую отдельную секцию с карточками статистики.
- **Секции с видео-фоном**: hero, Why Bali и Wanayu используют `<video autoplay muted loop playsinline>` с overlay div для читаемости текста
- **Декоративный водяной знак**: добавить `.logo-watermark .logo-watermark--right` или `.logo-watermark--left` к секциям для декоративного фона с лотосом. Модификатор `--large` для увеличения на 40%.
- **Именование компонентов**: БЭМ-подобное (`block__element--modifier`), напр. `.project-showcase__content`, `.project-card__badge--presale`
- **Все страницы имеют общие**: идентичные шапку (с CTA-кнопкой, языковым дропдауном), подвал (с SVG-иконками соцсетей) и плавающую кнопку WhatsApp — изменения нужно реплицировать на все **36 HTML-файлов** (9 EN + 9 RU + 9 ID + 9 ZH)
- **Порядок навигации**: Home, Projects, Services, About, Gallery, Contact + CTA-кнопка (открывает квиз) + языковой дропдаун
- **Языковой дропдаун**: кнопка `.header__lang` показывает текущий код языка (EN/RU/ID/中文) + шеврон SVG → выпадающий список из 4 языков с галочкой у активного. Логика в `main.js`. CSS: `.header__lang-toggle`, `.header__lang-dropdown`. Закрывается по клику вне или Escape.
- **Относительные пути в подпапках**: `ru/`, `id/`, `zh/` используют префикс `../` для CSS, JS, изображений, видео. Навигационные ссылки ведут на соседние страницы (напр. `index.html`, а не `../index.html`). Ссылки переключателя языков используют `../` для EN и `../ru/`, `../id/`, `../zh/` для остальных.
- **Иконки**: SVG line-art стиль (`stroke: var(--color-accent); stroke-width: 1.5; fill: none`) — используются в карточках услуг, фичах проектов и опциях квиза
- **Внешние ссылки**: всегда `target="_blank" rel="noopener noreferrer"`
- **Изображения**: организованы в `images/common/`, `images/home/`, `images/serenity-villas/`, `images/serenity-estates/`, `images/serenity-village/`. Все изображения ниже первого экрана используют `loading="lazy"`.
- **SEO**: каждая страница имеет `<title>`, `<meta description>`, OG-теги, Twitter Card теги, canonical ссылку и `<link rel="icon">`. Страницы проектов и контактов имеют JSON-LD structured data.
- **Availability bars**: карточки проектов и детальные страницы показывают прогресс-бары продаж юнитов. Village использует presale-вариант (пульсирующая точка, без прогресс-бара). Данные захардкожены в HTML.
- **ROI Калькулятор** (index.html): интерактивный с 3 сценариями (Консервативный/Нормальный/Оптимистичный), регулируемый слайдер заполняемости, CTA-кнопка открывает квиз.
- **Производительность**: обработчики скролла через `requestAnimationFrame`, все scroll-листенеры с `{ passive: true }`. Hero-постер предзагружен через `<link rel="preload">`. Декоративный водяной знак использует `logo-transparent.png` (33KB) вместо `LOGO.png` (120KB). Видео используют `-movflags +faststart` для прогрессивной загрузки.
- **Мобильное бургер-меню**: эффект матового стекла (`backdrop-filter: blur(24px)`, 75% непрозрачность фона), водяной знак лотоса на 30% непрозрачности.
